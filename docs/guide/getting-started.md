# Getting Started

Welcome to the IQToolkit ecosystem! This guide will help you understand how to discover and use the various tools available in the IQToolkit collection.

## What is IQToolkit?

IQToolkit is not a single package, but rather an ecosystem of specialized tools designed for data analysis, database optimization, and intelligent automation. Each tool is developed independently and can be used standalone or in combination with other tools.

## Prerequisites

Most IQToolkit tools require:

- **Python 3.11+** (Python 3.12+ recommended)
- **pip** for package management
- **Git** for accessing source code (optional)

## Installation Workflow

Each tool follows a similar installation pattern:

1. **Check requirements** in the tool's documentation
2. **Install the tool** using pip or your preferred method
3. **Configure** as needed for your environment
4. **Start using** the tool with examples and guides

## Example: Getting Started with a Tool

Here's the general workflow for any IQToolkit tool:

1. **Visit the tool documentation** (e.g., [Slow Query Doctor](/tools/slow-query-doctor/))
2. **Follow the tool's installation instructions** 
3. **Review the tool's configuration options**
4. **Run through the tool's getting started examples**

```bash
# General pattern (varies by tool):
pip install tool-name
tool-name --help
```

Each tool provides detailed examples and tutorials in its own documentation section.
