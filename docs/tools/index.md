# Tools

Welcome to the IQToolkit tools documentation. This section contains documentation for all the individual tools in the IQToolkit ecosystem.

## Available Tools

- **[Slow Query Doctor](/tools/slow-query-doctor/)** - Advanced SQL query performance analysis and optimization tool

## Contributing Tools

If you're building a tool for the IQToolkit ecosystem, please follow our [contributing guidelines](/contributing) to have it included in this documentation.

### Adding Your Tool's Documentation

1. Ensure your tool repository has a `/docs` folder with markdown documentation
2. Submit a pull request to add your repository to the sync configuration
3. Your docs will automatically be pulled and published here

## Tool Development Guidelines

- Follow our [API standards](/api/overview)
- Include comprehensive documentation in your `/docs` folder
- Use consistent naming and structure
- Provide examples and usage instructions
